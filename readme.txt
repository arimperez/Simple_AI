Program Description
####################
Simple AI Program that learns as you ask it questions given user provided rules. 

Program is given keywords and replies which are added to a binary search tree with an
importance value. The value decreases when the user deems the response unhelpful. 

Additonal Program  Details
####################

To run my program, make program, then run the executable
with a .txt file holding AI Rules

ALL rates are set to default before user changes them (0.5 for system and user rate, 0.05 for learning rate)

Assumption:
-All keywords are different

Binary Search Tree
######################################
Elements are put into the binary search tree by their keyword
